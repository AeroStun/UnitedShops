# Unitedshops
Bukkit 1.8 Admin Shop Plugin using Vault.
This is a Spigot 1.8 JavaPlugin. The goal is the creation of an admin shop with Vault integration.
With this plugin you can create Shops with GUIs. Shops are shown in a category GUI. Opening a shop shows a list of all items sold. You can click on an item to open the buy and sell GUI to buy/sell 1/10/64 items.
